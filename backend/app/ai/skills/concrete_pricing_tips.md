## 混凝土组价的核心决策树

### 1. 确定混凝土类型

- **商品混凝土（预拌）**：城市项目首选，含运输费，但需计**泵送费**或吊斗运输费。
- **现场搅拌**：偏远工地或小体量工程使用，须另计砂、石、水泥、搅拌机台班、人工。
- **自密实混凝土（SCC）**：用于密集钢筋或复杂模板部位，单价比普通 C30 高约 15-25%。

### 2. 选择正确的定额子目

**关键**：定额按**结构部位**区分，不能混用：
- 基础：独立基础 / 带形基础 / 满堂基础 / 桩基承台
- 柱：矩形柱 / 圆形柱 / 构造柱 / 异形柱
- 梁：矩形梁 / 圈梁 / 过梁 / 异形梁
- 板：有梁板 / 无梁板 / 平板 / 密肋板 / 楼梯板
- 墙：直形墙 / 弧形墙 / 挡土墙 / 电梯井壁

套错子目会导致人工费、机械费偏差 10-30%。

### 3. 强度等级与掺合料

- **C25 及以下**：一般用于垫层、构造部位。
- **C30**：主流结构强度，价格基准。
- **C35-C40**：高层或受力较大部位。
- **C50+**：特殊结构，须核对混凝土供应商报价，**不能按定额直接套**。

掺合料（粉煤灰、矿粉、外加剂）：
- 定额基价已含基础掺合料用量。
- 若招标文件要求**特殊耐久性要求**（如氯离子含量、抗硫酸盐），应在项目特征中注明并调增单价。

## 必须配套计量的项目

一条混凝土清单项通常需要**同时组价**以下配套项：

1. **模板（支模、拆模、周转）**：按 **m²** 展开面积。
2. **钢筋**：按 **t** 分直径、分等级。
3. **泵送 / 运输** ：商品混凝土需另计。
4. **养护**：冬季施工、蒸汽养护、喷淋养护另计。
5. **构造节点**：止水带、后浇带、施工缝处理。

## 易错点

1. **忘算泵送**：高层结构泵送费可达 C30 主料单价的 10-15%。
2. **模板面积错按净面积**：应按**与混凝土接触面**的展开面积。
3. **构造柱 / 圈梁归类错误**：构造柱不可计入框架柱定额，价差约 20%。
4. **楼梯工程量按水平投影 vs 斜长**：楼梯混凝土按**水平投影面积 × 0.4** 折算体积（或按图精算），模板按**与混凝土接触面**展开。
5. **后浇带**：独立计量，不计入主体。
6. **抗渗要求**（P6、P8、P10）：须调增单价或单列子目。

## 价格合理性检查（风向标）

混凝土综合单价（含模板、不含钢筋）常见区间（2023 参考，仅供快速核对，不替代报价）：

| 部位 | C25 | C30 | C35 | C40 |
|-----|------|------|------|------|
| 独立基础 | 500-650 | 550-700 | 600-750 | 650-800 |
| 柱 | 700-900 | 750-950 | 800-1000 | 850-1100 |
| 梁板 | 650-850 | 700-900 | 750-950 | 800-1000 |
| 墙 | 700-900 | 750-950 | 800-1000 | 850-1100 |
| 楼梯 | 900-1200 | 1000-1300 | - | - |

偏离 30% 以上须重点核查。**价格单位：元/m³**，含人工、材料、机械、管理费、利润。

## 区域系数（粗略）

- 一线城市（北上广深）：在全国平均基础 × 1.10-1.25
- 省会城市：× 1.00-1.10
- 三四线城市 / 县城：× 0.85-0.95
- 偏远地区：另加运输补贴，视项目

## 快速校核清单

审核混凝土组价时依次检查：
- [ ] 定额子目与结构部位匹配？
- [ ] 强度等级准确？（不混等级）
- [ ] 模板、钢筋、泵送、养护独立计量？
- [ ] 抗渗 / 特殊要求已体现？
- [ ] 构造柱 / 圈梁 / 过梁未错归入框架？
- [ ] 价格落在合理区间？（偏差 > 30% 须解释）
